## ⚡ Quick Start (3 Steps)

### 1. Verify Setup
```bash
./verify-macos.sh
```

### 2. Install Dependencies (if needed)
```bash
npm install
```

### 3. Start NovelCraft
```bash
npm run dev
```

## 🔧 Troubleshooting

### If Something Goes Wrong
```bash
# Clean everything and start fresh
npm run clean

# Verify your setup
npm run verify:macos

# Rebuild native modules
npm run rebuild
```

### Common Issues
- **Permission errors**: Run `chmod +x setup-macos.sh`
- **Build failures**: Install Xcode Command Line Tools
- **Native module issues**: Run `npm run rebuild`
